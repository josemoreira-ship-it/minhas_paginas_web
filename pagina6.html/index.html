O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar páginas HTML, definindo cores, fontes, layouts e muito mais.

Ele permite separar a estrutura (HTML) da apresentação (estilos), tornando o desenvolvimento mais organizado.

Tipos de Seletores CSS

Existem diversos tipos de seletores que permitem aplicar estilos de diferentes maneiras:

  • Seletor de elemento: aplica estilos a todas as tags especificadas.
  • Seletor de classe: aplica estilos a elementos que possuem determinada classe.
  • Seletor de ID: aplica estilos a um elemento específico que possui um ID único.
