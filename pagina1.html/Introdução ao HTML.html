Introdução ao HTML

O HTML (HyperText Markup Language) é a linguagem padrão utilizada para a construção de páginas na web. Ele não é considerado uma linguagem de programação, mas sim uma linguagem de marcação, responsável por estruturar os conteúdos que serão exibidos em um navegador. Com o HTML é possível organizar textos, imagens, links, tabelas, formulários e diversos outros elementos que compõem um site.

Um documento HTML é formado por tags (etiquetas), que são comandos utilizados para indicar como cada parte do conteúdo deve ser interpretada. As tags geralmente aparecem em pares, com uma abertura e um fechamento .

As principais tags que compõem a estrutura básica de uma página são:

• < !DOCTYPE html >: indica ao navegador que o documento está escrito em HTML5.

• < html >: marca o início e o fim do documento HTML.

• < title >: define o título exibido na aba do navegador.

• < body >: contém todo o conteúdo visível ao usuário, como textos, imagens e links.

Dentro do < body >, algumas tags importantes são:

• < h1 > a < h6 >: definem títulos e subtítulos, do mais importante (< h1 >) ao menos importante (< h6 >).

• < p >: cria parágrafos de texto.

• < a >: insere links para outras páginas ou endereços na web.

• < img >: adiciona imagens.

• < ul >, < ol > e < li >: criam listas, sendo não ordenadas (com marcadores) ou ordenadas (com números).

  • < br >: quebra de linha.
